# 问卷星导入格式生成器

将问卷内容生成为**问卷星文本导入格式**，用户可直接复制粘贴到问卷星平台中导入创建问卷。

## 触发条件

- 用户提到"问卷星"、"wenjuanxing"、"制作问卷"、"创建调查问卷"
- 用户要求以问卷星格式输出问卷内容
- 用户需要将已设计好的问卷转为问卷星可导入格式

## 文本格式规范

### 基本规则

1. **标题**：第一行为问卷标题，如 `【示例】大学生安全防范意识情况的调查问卷`
2. **题号**：每题必须有数字题号（`1.`, `2.`, ...），如无题号需在题目之前加空白换行
3. **题型标注**：在题目末尾用 `[题型名]` 标注题型，如 `[单选题]`、`[多选题]`

### 支持的题型

| 题型 | 标注 | 格式说明 |
|------|------|----------|
| 单选题 | `[单选题]` | 选项用 A. B. C. D. 列出，每行一个选项 |
| 多选题 | `[多选题]` | 同上，选项用字母或数字均可 |
| 量表题 | `[量表题]` | 题目下一行开始，每行列出一个量表等级 |
| 矩阵题 | `[矩阵题]` | 第一行是列标题（空格分隔），后续行为行标题 |
| 排序题 | `[排序题]` | 选项按行列出 |
| 比重题 | `[比重题]` | 选项按行列出 |
| 表格题 | `[表格题]` | 第一行：选项列标题（空格分隔），第二行：行标题的列标题（空格分隔），后续行为行标题 |
| 多项填空题 | `[多项填空题]` | 题目中使用 `____` 下划线表示填空位置 |
| 段落说明 | `[段落说明]` | 仅文字，不要求回答，用于分隔章节或说明 |

### 各题型的标准模板

**单选题/多选题：**
```
1.题目内容[单选题]
A.选项一
B.选项二
C.选项三
D.选项四
```

**量表题：**
```
1.题目内容[量表题]
很不满意
不满意
一般
满意
很满意
```

**矩阵题：**
```
1.题目内容[矩阵题]
经常 偶尔 没有
行标题一
行标题二
行标题三
```

**排序题：**
```
1.题目内容[排序题]
A.xxx
B.yyy
C.zzz
```

**表格题：**
```
1.题目内容[表格题]
选项1 选项2 选项3
列标题1 列标题2 列标题3
行标题1
行标题2
```

**多项填空题：**
```
1.姓名____, 年龄:____, 体重:____.
```

**段落说明：**
```
第一部分[段落说明]
```

## 输出流程

1. **理解需求**：确认问卷主题、目标受众、所需题型
2. **设计问题**：撰写清晰、中立的问题和选项
3. **按格式输出**：严格按照上述格式生成，确保每题有题号、题型标注正确
4. **检查**：确认格式符合问卷星导入要求：
   - 每道题都有数字题号
   - 题型标注正确放在 `[]` 中
   - 选项格式正确（字母或数字后跟选项文本）
   - 矩阵题的列标题用空格分隔
   - 填空题使用 `____` 下划线

## 完整示例

```
【示例】大学生安全防范意识情况的调查问卷
1.您的年级是[单选题]
A.2007级
B.2006级
C.2005级
D.2004级

2.请对学校的安全管理制度的满意度作个评价[量表题]
很不满意
不满意
一般
满意
很满意

3.请对学校的安全管理制度的满意度作个评价[矩阵题]
很不满意 不满意 一般 满意 很满意
寝室里面用电安全管理
食堂就餐安全管理

4.请按您认为的安全要素的重要程度排序[排序题]
用电安全
食堂就餐安全
公共安全

5.您觉得我们学校有哪些安全措施令您觉得很有安全感[多选题]
A.保安巡逻
B.监控摄像头
C.门禁系统
D.消防设施
```

## 注意事项

- 单/多选题选项建议使用字母编号（A. B. C. D.），也可以直接写选项文字
- 矩阵题第一行的列标题使用**空格**分隔，**不要用逗号或制表符**
- 填空题的下划线 `____` 建议连续4个以上，确保问卷星能正确识别
- 段落说明不需要题号，直接写文字标注即可
- 如果题目无题型标注，问卷星会自动智能识别，但推荐显式标注
